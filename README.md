# 카카오톡을 벤치마킹한 메신저 웹 서비스

- 평소 대중적인 메신저 서비스인 카카오톡을 사용하면서 읽은 메시지 처리 로직에 대해 매우 궁금증이 많았던 저희는 '카카오톡을 벤치마킹한 메신저 웹 서비스'를 개발하는 프로젝트를 진행하게 되었습니다.
- 개발 기간 : 2020-01 ~2020.02(약 2달)
